# tsreact-electron-template

Basic and lightweight Electron starter with React and Typescript

## Features

- React (unejected CRA)
- Typescript
- Electron Builder
- IPC with preload script
- Latest everything (Typescript 4+, React 17+, Electron 10+)
- Nothing more, nothing less
